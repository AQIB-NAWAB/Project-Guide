## Layout + Outlet mental model

```
Route: /vendor element=<VendorLayout>
  Child: open-shop → <OpenShop />     ← no gate redirect
  Child: settings  → <StoreSettings /> ← gate required
  Child: products  → <Stub />          ← gate required
```

`VendorLayout`:

1. If path is open-shop → render Outlet without gate (or gate only checks auth)  
2. Else → fetch GET me → 404 redirect open-shop  
3. Else → render nav + Outlet  

---

## Loading states

| Phase | UI |
|---|---|
| GET me pending | Full-layout spinner or skeleton |
| GET me 404 on gated route | Redirect — no flash of dashboard |
| POST pending | Disable submit + "Opening shop…" |

**❌ Don't:** Show empty dashboard for 200ms then redirect — jarring.
