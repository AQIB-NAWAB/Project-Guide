## Final ship checklist

### Runnable product

- [ ] Production client URL loads without errors
- [ ] Week 3 commerce demo works on **production** (browse → two vendors → cart → checkout → vendor sees order)
- [ ] Week 2 vendor flow works on production (login → products with photos)
- [ ] Week 4 search returns results on production
- [ ] Worker processed checkout on production (row 6 from 22.13)
- [ ] No console errors on primary user paths

### Repository hygiene

- [ ] README: local setup, deploy steps, demo accounts (no passwords — "see seed" or "contact mentor")
- [ ] `.env.example` files complete
- [ ] No secrets in Git history (`git log -- "**/.env"` empty)
- [ ] Sensible commit history — features findable

### Learning logs

- [ ] `learning-log/` contains completed files for chapters you finished
- [ ] Each gate chapter has honest answers — not one-line stubs

## Common beginner questions

**Q: Is Chapter 99 more coding?**  
Mostly **verification and packaging** — checklists, docs, video. No new features unless something is broken on production.

**Q: My Atlas cluster paused. Am I done?**  
Not until live URL works at evaluation time. Redeploy or wake cluster before submitting.

**Q: Do I need a custom domain?**  
No — platform HTTPS URLs are fine. Custom domain is polish.

**Q: What if I skipped optional sub-chapters?**  
Fine if gates are honestly met. Document skips in learning log — mentors prefer honesty over checkbox theatre.
