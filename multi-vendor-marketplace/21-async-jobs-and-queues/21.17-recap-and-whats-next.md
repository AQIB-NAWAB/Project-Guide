## Architecture you can explain

```
API commits commerce state synchronously
        ↓
API enqueues side-effect jobs (milliseconds)
        ↓
API returns 201/200
        ↓
Worker processes jobs independently
```

Week 4 demo: confirmation page **before** worker logs — that is correct behaviour.
