## Pagination UI spec

Choose one — document in README:

| Pattern | Implementation |
|---|---|
| Prev / Next | Disable Prev on page 1, Next on page === totalPages |
| Page numbers | 1 … totalPages — simpler for gate: Prev/Next only |
| Load more | Append items — **not** default; offset API expects page replace |

**Recommended for course:** Prev / Next + "Page X of Y" text using API `page` and `totalPages`.

Changing page scrolls to top optional stretch.

---

## Loading and empty

| Condition | UI |
|---|---|
| loading | Skeleton or "Loading…" |
| error | Error message + retry button |
| items.length === 0 && !loading | Empty state — adjust filters hint |
| success | Grid + pagination |

---

## Limit

Use `limit: 20` — match API default. Do not request 500 to avoid pagination.
