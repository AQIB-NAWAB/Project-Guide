## Troubleshooting before you leave Chapter 13

| Blocker | Where to look |
|---|---|
| 401 on upload | Ch 7 interceptor; multipart + Bearer |
| 403 unexpected | Ch 11 ownership; storeId chain |
| Multer "Unexpected field" | Field name must be `image` both sides |
| Empty imageUrl after 200 | Service save bug; check await |
| CORS on POST | Server CORS config for Vite origin |
| Huge list response | Embedded image data — fix schema/API |
| Cloudinary 401 | Wrong API secret in `.env` |
