## Definition of Done

### Stores module

- [ ] `server/src/modules/stores/stores.routes.js` exists and exports router
- [ ] `server/src/modules/stores/stores.service.js` — create, getByVendorId, update
- [ ] `server/src/modules/stores/stores.schema.js` — POST and PATCH validation
- [ ] Router mounted at `/api/stores` (or documented equivalent prefix)
- [ ] `GET /me` registered before `/:id` routes

### POST /api/stores

- [ ] Requires authentication — 401 without token
- [ ] Requires vendor role — 403 for customer
- [ ] Creates store with `vendorId` from JWT — not from body
- [ ] Returns 201 with store document on first create
- [ ] Second create same vendor → **409 Conflict**
- [ ] Invalid body → 400 with field errors

### GET /api/stores/me

- [ ] Vendor with store → 200 + document
- [ ] Vendor without store → **404** (or documented alternative)
- [ ] Customer → 403
- [ ] Query scoped by authenticated vendor only

### PATCH /api/stores/:id

- [ ] Owner can update mutable fields → 200
- [ ] Vendor B patching Vendor A store → **403**
- [ ] Invalid id or missing store → 404
- [ ] `vendorId` cannot be changed via PATCH
- [ ] Uses `assertStoreOwner` or equivalent from Ch 8
