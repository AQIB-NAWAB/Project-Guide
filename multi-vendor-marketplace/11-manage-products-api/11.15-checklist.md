## Definition of Done

### Products module

- [ ] `products.routes.js`, `products.service.js`, `products.schema.js` exist
- [ ] Router mounted; paths documented in README
- [ ] `GET /vendor/products` registered before `/:id` if shared router
- [ ] `assertProductOwner` (or equivalent) in shared ownership helpers

### POST /api/products

- [ ] Vendor + valid body → 201; `storeId` set server-side
- [ ] Vendor without store → 404 (or documented alternative)
- [ ] Customer → 403; no token → 401
- [ ] Invalid body → 400
- [ ] `storeId` in body ignored or rejected

### GET /api/vendor/products

- [ ] Returns only authenticated vendor's store products
- [ ] Includes unpublished/draft products
- [ ] Vendor B does not see Vendor A products
- [ ] Customer → 403

### PATCH /api/products/:id

- [ ] Owner → 200; changes persist
- [ ] Vendor B → **403**
- [ ] `isPublished` toggle works via PATCH
- [ ] `storeId` cannot change

### DELETE /api/products/:id

- [ ] Owner → 204 (or documented 200)
- [ ] Vendor B → **403**
- [ ] Product removed from list and Compass

### Testing & docs

- [ ] 11.13 test matrix executed — steps 5–6 are 403
- [ ] `docs/AUTHORIZATION.md` includes product routes
- [ ] Ch 9–10 gates still pass
