## Step 1 ÔÇö API contract

```
GET /api/vendor/orders
Authorization: Bearer <vendor-token>
```

**Middleware:** `requireAuth`, `requireRole('vendor')` ÔÇö Ch 18.14.

| Field | Use in UI |
|---|---|
| `_id` | Row key |
| `status` | Badge ÔÇö usually `pending` |
| `total` | Order total |
| `createdAt` | Sort/display date |
| `customer` / populated `customerId` | Name or email only |
| `orderItems[]` | Snapshot lines with `unitAtPurchase` |

**Scope:** Server resolves store via ownership ÔÇö **not** client `storeId` query param.
