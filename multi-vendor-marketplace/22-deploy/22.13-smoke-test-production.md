# 22.13 — Smoke test production

**Why this step matters:** Deployed is not done — smoke tests are the gate. Row 6 (worker log after checkout) catches the API-only deploy mistake that health checks alone never reveal.

**Where the project is now:** Client and API on HTTPS (22.10–22.12); env for API and worker (22.8); worker deployed — **must be running**.

**After this step:** Critical paths on **live HTTPS** verified; worker processes checkout job; results in README.

---

Prove the **deployed system** — API, client, MongoDB, Redis, Cloudinary, **and worker**. Health-only fails gate. Chapter 21.15 proved async locally — repeat on public URLs.

---

## Prerequisites

Worker running in host dashboard (not just API). Same `REDIS_URL` on both (22.8). Test customer `test+prod@...`. Incognito browser — no stale dev tokens.

---

## Smoke test script — mandatory

| # | Test | Pass criteria |
|---|---|---|
| 1 | `GET https://api.../health` | 200; `status: ok`, `db: connected` |
| 2 | Login customer | Works in **browser**; no CORS |
| 3 | Browse `/shop` | Grid from production API |
| 4 | Add to cart | Persists after refresh |
| 5 | Checkout | 201; order visible; cart cleared |
| 6 | **Worker smoke** | Worker logs job after checkout |
| 7 | Search | Results if wired (Ch 20) |
| 8 | Vendor login | Dashboard loads |

Rows 5–6 prove async pipeline. Row 6 fails on API-only deploy even when row 5 passes.

---

## Worker smoke test — contract

Verify worker **consuming** — not just Redis up or checkout 201.

| Method | Pass criteria |
|---|---|
| Logs | Tail worker **during** row 5 checkout |
| Pattern | `[NOTIFY]` from Ch 21.12 |
| Timing | Within ~30s of 201 |
| Multi-store | Two-store cart → multiple notify lines |
| Negative | Stop worker → 201, no log; restart → new checkout logs |

**No logs?** Check: worker service running; identical `REDIS_URL`; queue `notifications` (Ch 21); startup log present.

## Failure modes

| Symptom | Likely cause |
|---|---|
| Health OK, else fails | Wrong `VITE_API_URL` in client |
| 201, no worker log | Worker down; wrong `REDIS_URL` |
| Checkout 500 | MongoDB; stock; missing API env |
| Login browser-only fail | Cookie / CORS |
| Search empty | Redis index — Ch 20 |
