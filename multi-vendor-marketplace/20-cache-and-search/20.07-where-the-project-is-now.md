**Build cluster starts here.** Week 3 commerce works in browser. Every `GET /products` still hits MongoDB on every page load — fine for demo, wrong for scale. Chapter 20 adds **Redis cache-aside** on catalogue reads and a **text index** for search.

**After this chapter:** Identical browse UX with faster repeat loads; search uses index; cache busts on product write; API survives Redis stopped.
