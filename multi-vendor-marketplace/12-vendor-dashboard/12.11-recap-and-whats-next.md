# 12.11 — Recap and what's next

Chapter 12 completes the **vendor back office MVP** — product table, create and edit forms, publish toggle — all wired to Chapter 11 API. Vendors manage inventory in the browser without curl.

---

## Narrative recap

**products.js API helpers:** list, create, update (+ delete optional).

**ProductList:** GET /vendor/products table with status badges.

**CreateProduct:** POST with grocery fields — new items start as drafts.

**EditProduct:** PATCH updates including price and stock.

**Publish toggle:** PATCH isPublished — Draft/Published visible in list.

**Verification:** Full browser lifecycle documented.
