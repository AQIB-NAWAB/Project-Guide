# 7.6 — Prime your thinking

Before building auth pages, skim React patterns for context, protected routes, and fetch credentials.

## Why this matters for absolute beginners

Chapter 7 combines React Router, forms, cookies, and async boot logic — easy to glue wrong. Thirty minutes on Navigate, credentials, and your Chapter 6 curl notes prevents building pages that work in isolation but fail after reload or CORS.

## Common beginner questions

**Q: Why read fetch credentials docs for a React course?**  
A: Refresh cookies **require** `credentials: 'include'` — missing it looks like "refresh always 401."

**Q: Controlled inputs without a form library — really?**  
A: Yes for gate — fewer dependencies; focus on auth flow correctness.

**Q: CORS `*` with credentials?**  
A: Invalid combo — server must echo specific Vite origin with `Allow-Credentials: true`.
